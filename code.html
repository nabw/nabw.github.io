About Code Research

Here are links and small descriptions to the codes I use or used. I mainly work with Firedrake and Julia (Gridap and Ferrite).

RegFem
Image registration library. Its main focus is the study of mixed formulations for elastic registration.
FErosity
Library for the solution of nonlinear poromechanics problem.
Poroelasticity iterative
Library for testing block-splitting schemes for a three-field poroelastic model.
Poroelasticity preconditioners
Library for testing the effectiveness of splitting schemes as preconditioners.
lifex
Library used for the development of a complete human heart model (to be released soon).
Firdem
[WIP] Firedrake library used for computational cardiology. All in-house, write me an email for sharing.
AdaDIR
[WIP] Gridap library for image registration with parallel mesh adaptivity.
AAR.jl
[WIP] Julia library for efficient implementation of the Alternating Anderson-Picard method.

More side projects can be found in my Gitlab, Github and Bitbucket repositories. All codes I use are made available upon publication, but I will gladly share my work privately before that if reasonable, just send me an email.
